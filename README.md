# Lettzo Landing Page

This is the official landing page for **Lettzo**, the social meetup app for spontaneous, real-life connections.

## Tech Stack

- **Vite** - Build Tool
- **React** - UI Framework
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **shadcn/ui** - Component Library
- **Framer Motion** - Animations

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or pnpm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Production Build

To generate a production-ready build for hosting (e.g., on GoDaddy):

```bash
npm run build
```

The output will be in the `dist/` directory. You can upload the contents of the `dist/` folder to your GoDaddy hosting via FTP or File Manager.
